# Система учета абитуриентов 🎓

Это дипломный проект, разработанный для Академии ТОП. Система предназначена для автоматизации процесса работы с абитуриентами.

## 🚀 Основной функционал

- Учет и фильтрация данных абитуриентов.
- Ролевая модель доступа (Django Guardian).
- Динамические формы с автодополнением (DAL).
- Удобный интерфейс на Bootstrap 5.

## 🛡️ Безопасность и автоматизация

В проекте реализованы современные практики разработки:

- **Environment Variables**: Все секретные ключи и пароли вынесены в `.env` (используется `python-dotenv`).
- **Security Audit**: Интегрирована проверка уязвимостей библиотек через `pip-audit`.
- **Automation Scripts**:
  - `sync_reqs.py` — автоматическое обновление зависимостей и аудит безопасности.
  - `generate_env.py` — автоматическая генерация файла окружения с безопасными ключами.
  - `checkup.py` — комплексная проверка здоровья проекта (миграции, тесты, бэкапы).
- **Deployment Ready**: Настроены заголовки безопасности (HSTS, X-Frame-Options, CSRF/Session Cookie Secure).

## 🛠️ Стек технологий

- **Backend**: Python 3.12, Django 6.0
- **Database**: PostgreSQL
- **Frontend**: Bootstrap 5, Django Crispy Forms
- **Tools**: WhiteNoise (статика), Gunicorn (WSGI сервер)
